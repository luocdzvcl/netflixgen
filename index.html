📖 Hướng dẫn deploy Cloudflare Worker (5 phút)
1. Tạo tài khoản Cloudflare
Truy cập workers.cloudflare.com → đăng ký miễn phí → xác nhận email.
2. Tạo Worker mới
Dashboard → Workers & Pages → Create → chọn Create Worker → đặt tên bất kỳ → Deploy.
3. Paste code worker.js
Sau khi deploy → click Edit code → xoá toàn bộ code mặc định → paste nội dung file worker.js vào → Save & Deploy.
4. Lấy URL Worker
URL có dạng https://tên-worker.tên-bạn.workers.dev — copy URL đó paste vào ô Bước 1 ở trên.
5. Xong!
Paste cookie vào ô Bước 2 → nhấn Check Cookie.
⚡ Cloudflare Workers miễn phí: 100,000 request/ngày — đủ dùng cá nhân.
