📖 Hướng dẫn deploy Cloudflare Worker
1. Tạo tài khoản Cloudflare
Truy cập workers.cloudflare.com → đăng ký miễn phí.
2. Tạo Worker
Workers & Pages → Create → Start with Hello World! → đặt tên → Deploy.
3. Paste code
Click Edit code → Ctrl+A xoá hết → paste nội dung worker.js → Save & Deploy.
4. Copy URL và dán vào Bước 1
URL dạng https://tên.tên.workers.dev.
⚡ Miễn phí 100,000 request/ngày. NFToken được lấy tự động cho mọi tài khoản HIT.
